HTML(Hyper Text Markup Language)
- 페이지의 제목, 문단, 표, 이미지, 동영상 등 웹의 구조를 담당

CSS(Cascading Style Sheets)
- 실제 화면에 표시되는 방법(색상, 크기, 폰트, 레이아웃 등)을 지정해 콘텐츠를 꾸며주는 시각적인 표현(정적)을 담당

Javascript
- 콘텐츠를 바꾸고 움직이는 등 페이지를 동작시키는 동적 처리를 담당

크로스 브라우징
- 구동방식이 조금씩 다른 여러 브라우저에서 동일한 사용자 경험(같은 화면, 동작 등)을 줄 수 있도록 제작하는 기술, 방법

웹 브라우저 용어
- 창(Window) - 브라우저 실행 시 전체 구성
- 탭 - 브라우저 창을 구분해줌
- 주소창(Address bar) - 주소가 명시되는 부분
- 뷰포트(Viewport) - 웹사이트가 출력되고 렌더링되는 부분


특수 문자 기호 용어 정리
- ` : 백틱, 그레이브(Backtick, Grave)
- ~ : 틸드, 물결(Tilde)
- ! : 엑스클러메이션, 느낌표(Exclamation mark)
- @ : 앳, 골벵이(At sign)
- # : 샵 ,넘버 사인, 우물 정(Sharp, Number sign)
- $ : 달러(Dollar sign)
- % : 퍼센트(Percent sign)
- ^ : 캐럿(Caret)
- & : 엠퍼센드(Ampersand)
- * : 에스터리스크, 별표(Asterisk)
- - : 하이픈, 대시, 마이너스(Hyphen, Dash)
- _ : 언더스코어, 로대시, 밑줄(Underscore, Low dash)
- = : 이퀄, 동등(Equals sign)
- " : (더블)쿼테이션, 큰 따옴표(Quotation mark)
- ' : 아포스트로피, 싱클 쿼테이션, 작은 따옴표(Apostrophe)
- : : 콜론(Colon)
- ; : 세미콜론(Semicolon)
- , : 콤마, 쉼표(Comma)
- . : 피리어드, 닷, 점, 마침표(Period, Dot)
- ? : 퀘스천, 물음표(Question mark)
- / : 슬래시(Slash)
- \ : 백슬래시, 역슬래시(Backslash)
- | : 버티컬 바(Vertical bar)
- () : 퍼렌서시스, 소괄호, 괄호(Parenthesis)
- {} : 브레이스, 중괄호(Brace)
- [] : 브래킷, 대괄호(Bracket)
- <> : 앵글 브래킷, 꺽쇠괄호(Angle Bracket)
